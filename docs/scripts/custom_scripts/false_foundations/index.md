# False Foundations

False Foundations is an experimental script that challenges one core assumption:

> **That the game you are playing behaves the way you think it does.**

---

## 🧠 Script Identity

- **Difficulty:** Advanced
- **Style:** High uncertainty / structure disruption
- **Focus:** Adaptation, interpretation, endgame decision-making

---

## 🎯 Core Themes

- questioning game structure (Legion)
- questioning information (Vortox, Poisoner, Drunk)
- questioning identity (Fang Gu)
- questioning victory itself (Heretic)
