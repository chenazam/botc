# Overview (False Foundations)

False Foundations is built around a layered disruption model:

- **Structure disruption** → Legion
- **Information disruption** → Vortox, Poisoner, Drunk
- **Identity disruption** → Fang Gu
- **Victory disruption** → Heretic

---

## 🧠 Core Concept

Players must determine:

- whether the structure is normal
- whether information can be trusted
- whether the win condition applies

---

## 🎭 How This Script Feels

At first, the game appears:

- readable
- solvable
- familiar

Over time, players may experience:

- persistent ambiguity
- unstable worlds
- increasing doubt

## 🔄 Game Flow

### 🟢 Early Game

- information appears mostly normal
- players begin building worlds
- structure feels stable

---

### 🟡 Mid Game

- contradictions may or may not appear
- multiple explanations remain viable
- structure becomes questionable

---

### 🔴 Late Game

- Legion may prevent resolution
- Heretic may invert correct play
- Vortox may invalidate information
- final execution becomes highly unstable

---

## 🔍 Key Questions

Players should constantly ask:

- “Does this game behave normally?”
- “Can I trust this information?”
- “What happens if we are wrong?”

---

## ⚠️ Design Intent

This script is designed to:

- reward adaptive thinking
- punish overconfidence
- create meaningful doubt

---

## 🧠 Key Insight

> **False Foundations is not about solving faster — it is about realizing when your solution no longer applies.**
