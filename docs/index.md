# Blood on the Clocktower Role Bible

Welcome to the **Blood on the Clocktower Role Bible**.

This documentation is a personal strategy reference focused on:

- how to **play each role well**
- how to **bluff each role well**
- script-specific context and interactions
- practical advice for both newer and improving players

## What this site contains

### Roles

Detailed pages for individual roles, organized by:

- **script**
- **role type**

Each role page includes:

- overview
- playing the role as good
- bluffing the role as evil
- common mistakes
- advanced notes

### Scripts

Script-level reference pages covering:

- overall script identity
- setup dynamics
- common bluff environments
- important role interactions

### Reference

General strategy pages covering:

- bluffing basics
- good and evil fundamentals
- poisoning and drunkenness
- final three decisions
- setup logic
- glossary terms

## Suggested ways to browse

- Start with **Roles → By Script** if you want script-specific context
- Start with **Roles → By Type** if you want to compare similar roles
- Use **Reference** for broader concepts that apply across many roles

## Current focus

This documentation currently begins with:

- **Trouble Brewing**
- role-by-role pages
- a consistent format for both good play and evil bluffing

## Notes

This is a living reference and may evolve over time as more roles, scripts, and examples are added.
