# Three Ways to Solve a Game

Every Blood on the Clocktower script asks you to solve the game in a different way.

Understanding these differences is one of the biggest steps toward becoming a strong player.

---

## 🧠 Core Principle

> **Different scripts reward different ways of thinking — the best players adapt their approach accordingly.**

---

## 🎯 Why This Matters

Many players struggle when moving between scripts because they:

- apply the wrong mental model
- expect the game to behave the same way
- misinterpret information and events

Understanding _how_ a script is meant to be solved lets you:

- read the game more accurately
- avoid common mistakes
- adapt faster to new situations

---

# 🟢 Trouble Brewing — Build Worlds

Trouble Brewing is a script about **constructing logical worlds from reliable information**.

---

## 🧩 How it works

You:

- collect information
- build a consistent world
- eliminate worlds that break

---

## 🧠 What matters

- consistency
- setup logic
- outsider count
- role interaction

---

## ⚠️ Common failure

> Locking into a world too early and ignoring contradictions.

---

## 🎯 Summary

> **Solve the game by building the cleanest possible world.**

---

# 🟣 Sects and Violets — Manage Worlds

Sects and Violets is a script about **handling conflicting and unstable information**.

---

## 🧩 How it works

You:

- maintain multiple possible worlds
- track contradictions without resolving them immediately
- adapt as the game evolves

---

## 🧠 What matters

- ambiguity
- misinformation
- madness and social pressure
- flexibility

---

## ⚠️ Common failure

> Forcing clean logic onto a script that does not support it.

---

## 🎯 Summary

> **Solve the game by managing uncertainty better than your opponents.**

---

# 🔴 Bad Moon Rising — Interpret Events

Bad Moon Rising is a script about **understanding what actually happened during the night**.

---

## 🧩 How it works

You:

- analyze deaths and survival
- interpret layered interactions
- compare possible explanations

---

## 🧠 What matters

- death patterns
- protection layers
- timing and delayed effects
- overlapping causes

---

## ⚠️ Common failure

> Assuming outcomes have a single cause.

---

## 🎯 Summary

> **Solve the game by correctly interpreting what happened — and why.**

---

# 🔄 Putting It Together

Each script emphasizes a different skill:

- **Trouble Brewing** → build worlds
- **Sects and Violets** → manage worlds
- **Bad Moon Rising** → interpret events

---

## 🧠 Key Insight

> **The strongest players do not use one strategy — they switch between them depending on the script.**

---

# 🎯 Practical Heuristic

When entering a game, ask:

1. Is this a “clean information” script? → build worlds
2. Is this a “conflicting information” script? → manage worlds
3. Is this a “chaotic outcomes” script? → interpret events

---

# 🧠 Final Insight

> **You don’t solve every game the same way — you solve it the way the script demands.**
