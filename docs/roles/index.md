# Roles

This section contains the main **role reference pages**.

Each role page is the canonical home for that role’s strategy guide and includes:

- what the role does
- how to play it effectively as good
- how to bluff it effectively as evil
- common mistakes
- advanced notes and edge cases

Role pages are stored canonically **by script**, then grouped by **role type**.

The **By Type** pages are index pages that link to those canonical role files.
