# Roles by Type

This section provides an alternate way to browse role pages.

Instead of grouping roles by script, these pages group them by:

- Townsfolk
- Outsiders
- Minions
- Demons

This is useful when you want to:

- compare similar roles across scripts
- study one role category at a time
- identify recurring design patterns

## Notes

These pages are **index pages only**.

Each role’s actual guide is stored in its canonical location under:
